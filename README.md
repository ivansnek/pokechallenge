## PokeDesk Challenge
- Clone this repo: [PokeChallenge](https://github.com/ivansnek/pokechallenge)
- cd `pokechallenge`
- run `yarn` or `npm install` on the repo
- Then run `yarn start` or `npm run start` to execute the local server with the demo
